# 🪨📄✂️ Rock, Paper, Scissors 

An interactive, command-line implementation of the classic **Rock, Paper, Scissors** game built entirely in Python. Challenge the computer, track your score, and see who dominates!

---

## 🚀 Features

* **Smart Input Validation:** The game handles typos and invalid inputs gracefully, prompting you to enter a valid move.
* **Endless Play Loop:** Play as many rounds as you want; the game only stops when you decide to exit.
* **Real-time Score Tracking:** Keeps track of your wins, the computer's wins, and ties throughout your session.
* **Randomized AI:** The computer utilizes Python's `random` module to make unpredictable moves.

---

## 🎮 How to Play

1. **The Rules:**
   * 🪨 **Rock** beats Scissors ✂️
   * 📄 **Paper** beats Rock 🪨
   * ✂️ **Scissors** beats Paper 📄

2. **Run the Game:**
   Make sure you have Python installed, then run the script in your terminal:
   ```bash
   python rock_paper_scissors.py
